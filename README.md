## ✨ Features

- 👤 Gestión de usuarios
- 📦 CRUD de productos
- 🧾 Registro de ventas
- 🏢 Administración de proveedores y clientes
- 📂 Navegación intuitiva con React Router DOM
- 🎨 UI básica usando Bootstrap
- 🔍 Separación de vistas por módulos para cada entidad
